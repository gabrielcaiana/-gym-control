
#### Projeto - Sistema controle de academia

### Objetivo do projeto

- Cadastrar e consultar instrutores
- Cadastrar e consultar membros
- Editar instrutores e membros

#### Tecnologias utilizadas

- Node.js
- Vanilla.js
- Nunjucks
- HTML5
- CSS3

### Para executar o projeto

```
npm install
```

```
npm start
```
